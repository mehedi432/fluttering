# সবাইকে স্বাগতম ডার্ট প্রোগ্রামিং ল্যাঙ্গুয়েজ এর প্রথম পর্বে - আজকে আমরা আলোচনা করার চেষ্টা করবো ডার্ট প্রোগ্রামিং ল্যাঙ্গুয়েজ এর সংক্ষিপ্ত বিষয় নিয়ে - 

ডার্ট একটি Object Oriented প্রোগ্রামিং ল্যাঙ্গুয়েজ এর মানে হচ্ছে এই প্রোগ্রামিং ল্যাঙ্গুয়েজ অন্যান্য Object Oriented ল্যাঙ্গুয়েজ এর মতন এর সকল কিছুকেই এক একটি Object হিসাবে গণনা করে । এই ডার্ট প্রোগ্রামিং ল্যাঙ্গুয়েজ এর মধ্যে int 2 ও হচ্ছে একটি Object, এছাড়া Function এবং Null ও এক একটি Object । আমরা Object Oriented প্রোগ্রামিং সম্পর্কে আরও বিস্তারিত জানার চেষ্টা করবো পরবর্তী অধ্যায় গুলোতে ।
